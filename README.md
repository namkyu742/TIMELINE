## 프로젝트 참여
    장우영 : 프로젝트 총괄, 프로젝트 병합, 백업 및 복구 구현, 구글드라이브 연동 구현, 간편답장메모입력 구현, 논문 발표
    박채성 : 데이터베이스 구축, 비밀번호 구현, 메모 검색기능 구현
    조남규 : UI설계, 달력 기능 구현, 메모 삭제기능 구현, 보고서 작성
    양현모 : 이미지 사용 구현
